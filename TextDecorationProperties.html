Understanding CSS Text Decoration Properties

1. text-decoration-color

This text is underlined in red. (text-decoration-color)

2. text-decoration-line

This text has an overline in green. (text-decoration-line)

3. text-decoration-style

This text is underlined with a dashed blue line. (text-decoration-style)

4. text-decoration-thickness

This text has a purple line-through with 3px thickness. (text-decoration-thickness)

Combined Example

This text has both underline and overline, styled with a wavy orange line of 2px thickness. (Combined properties)
